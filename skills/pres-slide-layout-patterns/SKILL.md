# Slide Layout Patterns — 投影片版面模式庫

visual-designer 在製作投影片時使用的版面模式、格線系統與設計 Token 參考。

## 此技能的功能

- 提供 20 種投影片版面模式與適用情境
- 定義格線系統與安全邊距
- 提供色彩、字型、間距的設計 Token
- 提供無障礙設計檢查標準

## 適用時機

- 設計投影片版面時需要模式參考
- 建立簡報設計系統
- 搭配 `pres-presentation-designer` 技能使用

## 觸發範例

- 「這頁要放 3 個比較項目，用什麼版面？」
- 「標題頁怎麼設計比較好？」
- 「幫我定義這份簡報的設計系統」

## 格線系統（16:9 = 1920×1080px）

### 基礎設定

| 元素 | 數值 |
|------|------|
| 外邊距 | 80px（四邊） |
| 欄數 | 12 欄（欄寬 127px，間距 20px） |
| 列數 | 6 列（列高 153px，間距 20px） |
| 安全區域 | 距邊緣 120px（避免投影截切） |

### 垂直區域分配

| 區域 | Y 位置 | 高度 | 用途 |
|------|--------|------|------|
| 標題區 | 80px | 150px | 頁面標題 |
| 內容區 | 250px | 630px | 主要內容 |
| 頁腳區 | 900px | 100px | 頁碼、來源標注 |

## 20 種版面模式

### 封面類

**1. 標題頁**
- 主標題：56~72px，置中或左對齊
- 副標題：24~32px
- 背景：品牌色 / 全版圖片 + 色彩疊加
- 適用：簡報開場

**2. 章節分隔頁**
- 章節標題：48~56px，搭配章節編號
- 背景：品牌圖片或漸層
- 適用：多章節簡報的段落切換

### 文字類

**3. 大字報（Big Idea）**
- 單一句子：36~48px，置中
- 大量留白，視覺衝擊
- 適用：核心訊息強調、金句頁

**4. 文字 + 圖片（左右分割）**
- 左 6 欄文字 + 右 6 欄圖片
- 圖片：全出血，物件置中
- 適用：產品介紹、人物介紹

**5. 條列重點**
- 標題 + 3~5 個要點（超過 7 個需拆頁）
- 每點：icon（24px）+ 標題（20px bold）+ 說明（16px）
- 行距：1.6~1.8
- 適用：功能列表、步驟說明

### 數據類

**6. 關鍵數字（Key Metric）**
- 1~3 個核心數字：80~120px
- 單位與說明文字：18px
- 背景色塊強調
- 適用：KPI 展示、成果報告

**7. 兩項比較**
- 左右對稱結構
- 各自 icon + 標題 + 說明
- 適用：Before/After、競品比較、方案對比

**8. 三欄並列**
- 三個等寬欄位（最多四欄）
- 每欄：icon/圖片 + 標題 + 說明
- 適用：三大優勢、三個步驟、三種方案

### 時間軸類

**9. 時間軸（橫向）**
- 水平線 + 3~6 個節點
- 每節點：日期/階段 + 說明
- 適用：專案里程碑、歷史沿革

**10. 流程圖**
- 步驟方塊 + 箭頭連接
- 最多 5~6 個步驟（超過需拆頁）
- 適用：工作流程、決策流程

### 圖表類

**11. 單一圖表頁**
- 圖表佔 70% 版面
- 標題 + 圖表 + 來源標注
- 適用：數據深度分析

**12. 圖表 + 洞察**
- 左 7 欄圖表 + 右 5 欄文字洞察
- 洞察文字：3~4 個要點
- 適用：數據說故事

### 引用類

**13. 引言頁**
- 大引號視覺元素
- 引文：28~36px，斜體
- 來源：18px，右對齊
- 適用：客戶見證、名人引言

**14. 人物介紹**
- 圓形頭像 + 姓名 + 職稱 + 簡介
- 單人或多人（最多 4 人）
- 適用：團隊介紹、講者介紹

### 總結類

**15. 行動呼籲（CTA）**
- 核心訊息 + 明確行動項目
- 按鈕樣式的視覺強調
- 適用：簡報結尾、下一步說明

**16. 摘要頁**
- 3~5 個關鍵要點回顧
- 搭配頁碼參照
- 適用：章節總結、全簡報結尾

### 特殊類

**17. 全版圖片**
- 圖片全出血
- 文字疊加（需確保對比度）
- 適用：情境渲染、視覺衝擊

**18. 空白過場頁**
- 極簡設計，僅有品牌元素
- 適用：Q&A 頁、休息頁

**19. 目錄頁**
- 章節列表 + 頁碼
- 可加入視覺進度指示
- 適用：長篇簡報導覽

**20. 附錄頁**
- 較密集的資訊排版
- 字型可縮小至 14px
- 適用：補充數據、參考資料

## 設計 Token

### 字型系統

| 層級 | 大小 | 字重 | 用途 |
|------|------|------|------|
| Display | 56~72px | Bold | 封面主標題 |
| H1 | 40~48px | Bold | 頁面標題 |
| H2 | 28~36px | SemiBold | 副標題、章節標題 |
| H3 | 20~24px | SemiBold | 項目標題 |
| Body | 16~18px | Regular | 內文說明 |
| Caption | 12~14px | Regular | 來源標注、頁腳 |
| Metric | 80~120px | Bold | 關鍵數字 |

### 間距系統（8px 基準）

| 名稱 | 數值 | 用途 |
|------|------|------|
| xs | 8px | 元素內部間距 |
| sm | 16px | 相關元素間距 |
| md | 24px | 段落間距 |
| lg | 40px | 區塊間距 |
| xl | 64px | 主要區域間距 |

### 色彩規範

**主色（Primary）**：品牌主色，用於標題、強調元素、CTA

**輔助色（Secondary）**：主色的補色或類比色，用於次要強調

**中性色（Neutral）**：
- 文字：#1A1A1A（深）、#4A4A4A（中）、#8A8A8A（淺）
- 背景：#FFFFFF（白）、#F5F5F5（淺灰）、#E8E8E8（灰）

**語意色**：
- 成功：#22C55E
- 警告：#F59E0B
- 錯誤：#EF4444
- 資訊：#3B82F6

## 無障礙設計檢查

| 項目 | 標準 |
|------|------|
| 文字對比度 | 最低 4.5:1（WCAG AA） |
| 大文字對比度 | 最低 3:1（18px+ 或 14px+ Bold） |
| 色盲友善 | 不單靠顏色傳達資訊，搭配形狀或標籤 |
| 最小字型 | 投影片 16px，列印版 10px |
| 圖表色彩 | 避免紅綠並用，使用色盲模擬工具驗證 |
