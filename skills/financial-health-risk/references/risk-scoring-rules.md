# 風險評分規則

本文件定義財務健康風險的評分邏輯與閾值，確保分析結果一致可重現。

---

## 評分架構

### 維度權重

| 維度 | 權重 | 說明 |
|------|------|------|
| 獲利能力 | 25% | 企業核心競爭力的體現 |
| 流動性 | 20% | 短期償債能力，直接影響存活 |
| 槓桿／償債 | 20% | 長期財務結構穩健性 |
| 現金流 | 20% | 盈利品質與實際現金產生能力 |
| 成長性 | 15% | 未來發展潛力（數據不足時降權） |

**綜合風險分數** = Σ（各維度分數 × 權重）

---

## 各維度評分規則

### 流動性評分（滿分 100）

| 指標 | 權重 | 評分規則 |
|------|------|---------|
| 流動比率 | 50% | ≥ 2.0 → 100｜1.5~2.0 → 80｜1.0~1.5 → 60｜0.8~1.0 → 30｜< 0.8 → 0 |
| 速動比率 | 35% | ≥ 1.5 → 100｜1.0~1.5 → 80｜0.8~1.0 → 60｜0.5~0.8 → 30｜< 0.5 → 0 |
| 現金比率 | 15% | ≥ 0.5 → 100｜0.2~0.5 → 80｜0.1~0.2 → 60｜< 0.1 → 30 |

### 獲利能力評分（滿分 100）

| 指標 | 權重 | 評分規則 |
|------|------|---------|
| ROE | 30% | ≥ 20% → 100｜15~20% → 85｜10~15% → 70｜5~10% → 50｜0~5% → 25｜< 0% → 0 |
| ROA | 25% | ≥ 10% → 100｜7~10% → 85｜5~7% → 70｜2~5% → 50｜0~2% → 25｜< 0% → 0 |
| 淨利率 | 25% | ≥ 15% → 100｜10~15% → 85｜5~10% → 70｜2~5% → 50｜0~2% → 25｜< 0% → 0 |
| 毛利率 | 20% | 依行業基準比較：優良 → 100｜正常 → 70｜警示 → 30 |

### 槓桿／償債評分（滿分 100）

| 指標 | 權重 | 評分規則 |
|------|------|---------|
| 負債比率 | 40% | ≤ 30% → 100｜30~45% → 80｜45~60% → 60｜60~70% → 30｜> 70% → 0 |
| D/E Ratio | 35% | ≤ 0.5 → 100｜0.5~1.0 → 80｜1.0~1.5 → 60｜1.5~2.0 → 30｜> 2.0 → 0 |
| 利息保障倍數 | 25% | ≥ 10x → 100｜5~10x → 85｜3~5x → 70｜1.5~3x → 40｜< 1.5x → 0｜N/A → 90 |

### 現金流評分（滿分 100）

| 指標 | 權重 | 評分規則 |
|------|------|---------|
| CFO/Sales | 40% | ≥ 15% → 100｜10~15% → 80｜5~10% → 60｜2~5% → 30｜< 2% → 0 |
| FCF | 35% | 正且成長 → 100｜正且持平 → 75｜正但下滑 → 55｜負（擴張期）→ 40｜持續負值 → 10 |
| FCF/Net Income | 25% | ≥ 1.2 → 100｜0.8~1.2 → 85｜0.5~0.8 → 60｜0.2~0.5 → 30｜< 0.2 → 0 |

### 成長性評分（滿分 100）

| 指標 | 權重 | 評分規則 |
|------|------|---------|
| 營收 YoY | 50% | > 20% → 100｜10~20% → 85｜5~10% → 70｜0~5% → 50｜-5~0% → 25｜< -5% → 0 |
| 淨利 YoY | 30% | > 20% → 100｜10~20% → 85｜5~10% → 70｜0~5% → 50｜< 0% → 20 |
| 資產 YoY | 20% | 與營收成長匹配 → 80｜資產成長 > 營收成長 → 50｜資產萎縮 → 40 |

## Altman Z-Score 評分轉換

Z-Score 結果轉換為 0~100 分：

**上市公司版（Z-Score）**：
| Z 值 | 分數 | 說明 |
|------|------|------|
| > 3.5 | 100 | 財務極為健康 |
| 2.99 ~ 3.5 | 85 | 安全區上段 |
| 2.5 ~ 2.99 | 70 | 安全區下段 |
| 1.81 ~ 2.5 | 45 | 灰色地帶 |
| 1.2 ~ 1.81 | 20 | 危險區上段 |
| < 1.2 | 0 | 危險區，破產風險高 |

**非上市公司版（Z'-Score）**：
| Z' 值 | 分數 |
|-------|------|
| > 3.0 | 100 |
| 2.9 ~ 3.0 | 85 |
| 2.0 ~ 2.9 | 65 |
| 1.23 ~ 2.0 | 40 |
| < 1.23 | 0 |

Z-Score 分數納入「槓桿/償債」維度，作為額外加權參考（不替換原有指標，而是調整最終分數 ±5 分）。

## 綜合風險等級

| 綜合分數 | 風險等級 | 燈號 | 說明 |
|---------|---------|------|------|
| 80 ~ 100 | 低風險 | 🟢 | 財務健康，無明顯風險 |
| 60 ~ 79 | 中風險 | 🟡 | 部分指標需關注，建議改善 |
| 40 ~ 59 | 高風險 | 🔴 | 多項指標異常，需立即行動 |
| 0 ~ 39 | 危急 | ⛔ | 財務危機風險高，需緊急介入 |

---

## 數據缺失處理規則

| 缺失情況 | 處理方式 |
|---------|---------|
| 單一指標缺失 | 該指標給予中性分數（50），並在報告中標注 |
| 整個維度缺失（如無現金流量表） | 該維度給予 50 分，並降低報告信心度 |
| 成長性數據缺失（僅單期） | 成長性維度權重降為 0%，其他維度等比例提升 |
| 利息費用為 0 | 利息保障倍數標記 N/A，給予 90 分（無有息負債為正面信號） |
| 淨利為負 | ROE/ROA 給予 0 分，FCF/Net Income 標記異常 |

**報告信心度**：
- 三張報表齊全：高信心度（High Confidence）
- 缺少現金流量表：中信心度（Medium Confidence）
- 僅有損益表或資產負債表：低信心度（Low Confidence），報告需加強免責說明

---

## 特殊情況旗標

以下情況需在報告中特別標注，不論分數高低：

| 旗標 | 觸發條件 | 說明 |
|------|---------|------|
| 🚨 流動性危機 | 流動比率 < 1.0 且現金比率 < 0.1 | 短期償債能力嚴重不足 |
| 🚨 盈利品質差 | FCF/Net Income < 0.3 且淨利 > 0 | 帳面獲利但現金未跟上，可能有應收帳款問題 |
| 🚨 高槓桿警示 | 負債比率 > 70% 且利息保障倍數 < 2x | 財務槓桿過高，利息壓力大 |
| 🚨 Z-Score 危險 | Z-Score < 1.81（上市）或 Z' < 1.23（非上市） | 破產預測模型顯示高風險 |
| ⚠️ 成長停滯 | 營收連續 2 年 YoY < 0% | 業務萎縮，需關注競爭力 |
| ⚠️ 獲利惡化 | 淨利率連續下滑超過 3 個百分點 | 成本壓力或定價能力下降 |
