# 行業財務基準值參考

本文件提供各產業的財務指標基準值，用於相對比較分析。數據來源為台灣上市櫃公司財報統計與國際通用基準，每年應視市場狀況調整。

## 使用說明

1. 根據公司所屬產業選取對應基準表
2. 若產業無法明確分類，使用「國際通用基準」
3. 金融業（銀行、保險）財務結構特殊，不適用一般基準，需單獨處理
4. 基準值為中位數，非絕對標準，需結合公司發展階段判斷

## Altman Z-Score 行業調整說明

Z-Score 原始模型針對製造業設計，非製造業使用時需注意：

| 產業類型 | 建議版本 | 說明 |
|---------|---------|------|
| 上市製造業 | 原始 Z-Score | 使用市值計算 X4 |
| 非上市製造業 | Z'-Score | 使用帳面值計算 X4 |
| 服務業 | Z''-Score（Altman 1995）| 去除 X5（資產周轉率），避免服務業偏差 |
| 金融業 | 不適用 | 財務結構特殊，Z-Score 不具參考意義 |

**Z''-Score（服務業版）**：
```
Z'' = 6.56×X1 + 3.26×X2 + 6.72×X3 + 1.05×X4
```
- Z'' > 2.6：安全區 🟢
- 1.1 < Z'' ≤ 2.6：灰色地帶 🟡
- Z'' ≤ 1.1：危險區 🔴

## 成長性基準

| 成長率 | 評估 |
|--------|------|
| 營收 YoY > 15% | 高成長 🟢（需確認獲利同步成長）|
| 營收 YoY 5% ~ 15% | 穩健成長 🟢 |
| 營收 YoY 0% ~ 5% | 停滯 🟡 |
| 營收 YoY < 0% | 衰退 🔴 |
| 連續 2 年衰退 | 結構性問題 🔴 需深入分析 |
